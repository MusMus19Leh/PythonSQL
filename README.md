# FOP Pet Shop - Sistema de Gerenciamento Veterinario

Sistema desktop para gerenciamento de clinicas veterinarias, desenvolvido em Python com interface grafica e banco de dados PostgreSQL.

---

## Tecnologias Utilizadas

| Tecnologia   | Funcao                          |
|--------------|---------------------------------|
| Python 3     | Linguagem principal             |
| Tkinter      | Interface grafica desktop       |
| PostgreSQL   | Banco de dados relacional       |
| psycopg2     | Conexao Python com PostgreSQL   |
| Matplotlib   | Geracao de graficos             |

---

## Funcionalidades

### Sistema de Gerenciamento
- Login com autenticacao por usuario e senha
- Cadastrar, listar, atualizar e deletar clientes
- Cadastrar, listar e deletar pets vinculados a clientes
- Cadastrar, listar, atualizar e deletar consultas medicas

### Dashboard Analitico
- KPIs: total de consultas, receita total, ticket medio, clientes e pets atendidos
- Graficos de barras, pizza e linha com dados em tempo real
- Filtros dinamicos por especie e por mes
- Tabela detalhada com JOIN entre clientes, pets e consultas
